Display images in grid on searching and display image details on clicking each image.

- Frameworks used
  - MVVM architecture
  - Jetpack Compose
  - Hilt for Dependency Injection
  - Moshi for serialization
  - Retrofit
  - Kotlin Coroutines
  - MockK for Unit testing
  - Compose UI testing
  - Markdown Text (Markwon library)
